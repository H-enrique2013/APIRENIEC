API CONSULTA RENIEC


Url del Endpoint-Consulta Individual

File HTML: https://apireniec-am5e.onrender.com/consultaxDNINombresApellidos


Parámetros para el método POST

JSON de ejemplo: 

            {   "Typeconsulta": "0",  // El 0 es para una consulta por DNI
                "N_DNI": "12345678",  // Número de DNI
                "Nombres": "CARLOS",  // Nombre del usuario
                "Ap_Paterno": "SANCHEZ",  // Apellido paterno
                "Ap_Materno": "ROJAS"  // Apellido materno
            }
